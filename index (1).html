What I've Built

PROJECT_01
GAT-Based EEG Emotion Recognition for BCI Applications
GitHub Repo
ProblemEmotion recognition from EEG signals is complex — brain signals carry rich relational structure between electrodes that traditional models fail to capture.
What I DidBuilt an EEG-based system using Graph Attention Neural Networks (GAT), modelling electrode relationships as graph data with attention-based feature learning for BCI applications.
ResultAchieved ~97% accuracy — demonstrating that graph-based attention mechanisms significantly outperform conventional approaches for EEG classification.
Tech StackPython · PyTorch Geometric · Graph Attention Networks · EEG Signal Processing · Deep Learning
PROJECT_02
Credit Risk Analyzer — Loan Default Prediction
GitHub Repo
ProblemFinancial institutions struggle to accurately predict loan defaults — poor predictions lead to bad loans and significant financial losses.
What I DidIntegrated multiple datasets using SQL, performed end-to-end preprocessing in Python, and trained an XGBoost model. Added SHAP explainability to interpret key risk factors.
ResultAchieved ~95% accuracy with fully interpretable outputs — making predictions both accurate and explainable for real-world deployment.
Tech StackPython · SQL · XGBoost · SHAP · Pandas · Scikit-learn
PROJECT_03
RAG Search Assistant
GitHub Repo
ProblemLLMs hallucinate on domain-specific queries without grounded knowledge — making them unreliable for real-world information retrieval.

ResultDemonstrated efficient knowledge retrieval and significantly improved answer relevance over vanilla LLM responses for real-world queries.
Tech StackPython · LangChain · FAISS · LLMs · Semantic Search · Vector Embeddings
